# Interview Experiences Repository

Welcome to my Interview Experiences repository! This repository contains detailed accounts of my technical interview processes with various companies. Each subdirectory is dedicated to a specific company and includes relevant materials such as problem statements, code solutions, and interview questions.

## 📁 Repository Structure

```
Interview-experiences/
├── 1.DotNitron_Technologies (remote)/
│   ├── InterviewExperience.md / Interview_Experience.md / interview_experience.txt
│   ├── JobDescription.md / jobDescription.md / job_description.txt
│   └── (optional) Assignment.md / ProblemStatement.txt / code files

    ├── InterviewExperience.md
    └── jobDescription.md
```

## 📝 How to Navigate

1. Each company's interview experience is stored in its own directory, prefixed with a number for ordering.
2. The directory name follows the format: `[Number].[CompanyName] ([WorkLocation])`
   - Example: `1.DotNitron_Technologies (remote)`
3. Common files you'll find in each company's directory:
   - `InterviewExperience.md` / `Interview_Experience.md` / `interview_experience.txt` - Detailed notes about the interview process and questions
   - `JobDescription.md` / `jobDescription.md` / `job_description.txt` - The job description for the position
   - Some folders may include additional files like `Assignment.md`, `ProblemStatement.txt`, or code samples

## 🚀 Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/singhHarcharan/Interview-Experiences.git
   ```
2. Navigate to the specific company's directory you're interested in.
3. Review the interview experience and job description files.
