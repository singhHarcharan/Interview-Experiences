## About the Job

### About the Role

We are looking for a Forward Deployed Full Stack Engineer who can build and deploy production systems quickly while working closely with internal teams and customers.

This role combines full-stack development, system integration, and rapid problem-solving. You will work on real deployments, build custom features when needed, and integrate systems across different environments.

You should be comfortable working across frontend, backend, APIs, databases, messaging systems, and cloud infrastructure, and be open to using AI-assisted development tools to accelerate development.

### Key Responsibilities

#### Full Stack Development

- Build and maintain web applications using React.js, Next.js, and TypeScript.
- Develop backend services and APIs using Node.js and NestJS.
- Design and manage databases such as MongoDB or PostgreSQL.
- Build scalable APIs and integrate external services.

#### Messaging & Real-Time Systems

- Implement background jobs and queues using Redis and BullMQ.
- Build event-driven systems and asynchronous workflows.
- Implement real-time features using WebSockets or similar technologies.

#### Deployment & System Integration

- Deploy and manage applications on Microsoft Azure.
- Integrate with third-party APIs and external systems.
- Troubleshoot integration issues and production environments.
- Build solutions that operate reliably in real-world deployments.

#### Customer-Facing Engineering

- Work with internal teams and customers to understand requirements.
- Build custom workflows and integrations when needed.
- Prototype and ship solutions quickly to solve operational problems.

#### AI-Assisted Development

Use modern development tools such as:

- Claude Code
- OpenAI Codex
- Cursor
- GitHub Copilot

AI-assisted development is encouraged to increase speed, productivity, and experimentation.

#### Engineering Collaboration

- Participate in technical discussions and architecture decisions.
- Write clean, maintainable, and scalable code.
- Review code and contribute to engineering best practices.

### Required Skills

- 1+ years of development experience.
- Strong knowledge of JavaScript (ES6+) and TypeScript.
- Experience with React.js.
- Experience with Next.js.
- Backend development experience using Node.js and NestJS.
- Experience building and integrating REST APIs.
- Familiarity with MongoDB, PostgreSQL, or similar databases.
- Experience with Redis or similar in-memory data stores.
- Experience working with messaging queues such as BullMQ or similar.
- Familiarity with real-time communication technologies such as WebSockets.
- Strong debugging and problem-solving skills.
- Experience building full-stack applications.

### Good to Have

- Experience using AI-assisted development tools.
- Familiarity with LLM APIs or AI integrations.
- Experience with Docker.
- Experience deploying applications on Microsoft Azure.
- Experience integrating multiple external systems or APIs.
- Experience building scalable asynchronous systems.

### What We Look For

- Engineers who enjoy building systems end-to-end.
- Strong problem-solving and ownership mindset.
- Ability to move quickly and ship working solutions.
- Comfort working across frontend, backend, messaging systems, and integrations.
- Interest in using AI tools to accelerate development.
- Please include links to GitHub, portfolio, or projects you have built.

## Experience Requirement

Minimum: 1+ year of development experience.

Startup experience is encouraged.
