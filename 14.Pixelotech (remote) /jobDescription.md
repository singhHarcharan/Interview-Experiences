## Role overview

We’re hiring a Full Stack Engineer to ship end-to-end features across our AI native SaaS product with a strong focus on high-velocity, high-quality delivery  with modern AI coding tools (Cursor, Claude Code, etc.). You’ll work across frontend, backend, and AI-powered workflows (LLM + RAG) — with real production standards.

## What you’ll do

### Full-stack product engineering

- Build and maintain end-to-end features across frontend + Python backend services
- Design clean APIs, implement business logic, and evolve data models safely
- Write tests where they matter and keep releases stable (not hero debugging)
- Improve performance, reliability, and security across the stack
- Debug production issues with strong root-cause discipline

### AI feature engineering (LLM apps)

- Integrate LLM workflows into product surfaces (answering, extraction, summarization, classification, RAG)
- Improve quality via pragmatic evaluation (golden sets, regression checks, failure tracking)
- Optimize latency/cost/quality tradeoffs for real customer use

### Using AI coding tools (Cursor / Claude Code) as a force multiplier

You will use AI tools daily to increase throughput while maintaining engineering standards:

- Break work into tight tasks, write clear acceptance criteria, and drive execution fast
- Use AI for scaffolding + iteration, but ensure final code is clean, consistent, and correct
- Catch “AI slop”: hallucinated APIs, wrong assumptions, dead code, over-abstraction, missing tests
- Produce PRs that are readable, reviewable, and easy to maintain
